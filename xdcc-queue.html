XDCC-Queue 0.1

README

XDCC-Queue is a plugin for X-Chat written in C. It makes it easier to download from XDCC Bots by allowing users to present a range of packs to download, and XDCC-Queue will queue up those files for you, and download them sequentially.

Usage:

/start - Starts file sending process, automatically starts in this mode.

/stop - Stops file sending process. In order to continue file-queueing and sending, the user must issue the /start command.

/xdcc-queue BOTNAME FPACK LPACK - Queues up files from BOTNAME from FPACK to LPACK. For example, "/xdcc-queue GT_Onizuka 1 14" would queue up and start sending packs #1 - #14 from the bot named GT_Onizuka.

/help xdcc-queue - Presents general usage information from within X-Chat.

Installation:

*nix

See X-Chat *nix/Mac OS X plugin compile commands

Windows:

See X-Chat Windows plugin compile guide

After plugin has been successfully compiled, simple place the the shared object library file or DLL file in your .xchat2 directory so the plugin will be loaded at startup.
